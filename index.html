About this project

This project is a visual presentation of the WeatherPy project. The object of the project was to evaluate weather patterns as you approach the equator. To get this done I done used:

  • Python
  • Openweather API
  • Citipy
  • Matplotlib
  • Pandas
Steps:
  1. First I downladed a CSV file of all the cities covered by the WeatherAPI, there were lots.
  2. Next I picked 500 random cities from this list
  3. I used the WeatherAPI to get the latitude, temperature, cloudiness, humidity, and windspeed for each city.
Click on the visualizations below to see the results, each visualizaiton has a page with an explanation.
